=== MOD NAME (Nexus "Name" field) ===
FindMyRide - Vehicle Search and Filter

=== SHORT SUMMARY (Nexus "Summary" field, max 250 chars) ===
Adds text search and category filters (All, Favorites, Cars, Motorcycles, Armed) directly inside the native vehicle call menu (hold V). Type to filter, arrows to switch category. No extra windows. English and PT-BR.

=== DESCRIPTION (Nexus "Description" field, BBCode) ===
[size=4][b]FindMyRide - Vehicle Search and Filter[/b][/size]

Adds a [b]search bar and category filters directly inside the native vehicle call popup[/b] (hold [b]V[/b]) - no extra overlay, no separate window. Useful when you have many vehicles unlocked and the summon list gets long.

[size=3][b]✨ Features[/b][/size]
[list]
[*][b]Text search[/b] - press [b]Tab[/b] and type part of the name ("911", "quadra", "kusanagi"...). The list filters live with every keystroke. Smart matching: "type66" finds "Type-66"
[*][b]Category filters[/b] - cycle with [b]Left/Right arrows[/b] (or D-Pad on gamepad): All / Favorites / Cars / Motorcycles / Armed
[*][b]Live counter[/b] - always shows how many vehicles match the current filter
[*][b]Languages[/b] - English by default, Português (Brasil) automatically when the game interface is set to PT-BR
[/list]

[size=3][b]📖 How to use[/b][/size]
[list]
[*]Hold [b]V[/b] to open the vehicle call menu as usual
[*][b]Tab[/b] - enter/leave search mode. While searching, letters and numbers type into the search (list navigation is paused), [b]Backspace[/b] deletes, [b]Enter[/b] confirms the filter, [b]Esc[/b] clears it
[*][b]Left / Right arrows[/b] (or D-Pad left/right) - switch category at any time
[*][b]Esc[/b] outside search mode - clears filters step by step before closing the menu
[/list]

[size=3][b]📦 Installation[/b][/size]
Extract the archive into the game's root folder (the one that contains [i]bin[/i], [i]archive[/i], [i]r6[/i]...), or install with Vortex / Mod Organizer 2. The file lands in:
[code]r6/scripts/FindMyRide/FindMyRide.reds[/code]

[size=3][b]✅ Compatibility[/b][/size]
Game version 2.x. The mod wraps the native vehicle popup methods (no file patching, no replaced assets), so it is compatible with virtually everything, including vehicle unlock mods. Only a mod that fully replaces the vehicle call popup could conflict. Works on Windows and Linux/Proton. Safe to add or remove at any time.

Note: search typing uses physical key positions (US layout); accented characters are not needed since vehicle names are plain ASCII.

[size=3][b]🗑️ Uninstall[/b][/size]
Delete [code]r6/scripts/FindMyRide[/code]. Nothing else is touched.
