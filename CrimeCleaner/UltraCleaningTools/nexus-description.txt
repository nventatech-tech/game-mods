=== MOD NAME (Nexus "Name" field) ===
Ultra Cleaning Tools

=== SHORT SUMMARY (Nexus "Summary" field, max 250 chars) ===
Bigger cleaning area, sprays and cleaning liquids never run out, infinite power washer water (backpack too) and infinite trash bag capacity. Clean the whole scene without refilling or emptying anything.

=== DESCRIPTION (Nexus "Description" field, BBCode) ===
[size=4][b]Ultra Cleaning Tools[/b][/size]

Supercharges every cleaning tool in Crime Scene Cleaner so you spend your time cleaning, not managing supplies.

[size=3][b]✨ Features[/b][/size]
[list]
[*][b]Bigger cleaning area[/b] - tools clean a much larger radius per swipe (8x)
[*][b]Infinite cleaning liquids[/b] - sprays and chemicals never run out
[*][b]Infinite power washer[/b] - station and backpack water never deplete
[*][b]Infinite trash bags[/b] - trash capacity never fills up
[/list]

[size=3][b]📋 Requirements[/b][/size]
[list]
[*][url=https://melonwiki.xyz/]MelonLoader[/url] installed in the game folder
[/list]

[size=3][b]📦 Installation[/b][/size]
[list=1]
[*]Install MelonLoader in the game folder and run the game once
[*]Copy [b]UltraCleaningTools.dll[/b] into:
[code]Crime Scene Cleaner/Mods/UltraCleaningTools.dll[/code]
[*]Play
[/list]

[size=3][b]Notes[/b][/size]
[list]
[*]The trash patch waits a few seconds after each scene load to avoid interfering with loading - this is intentional
[*]No game files are modified - safe to add or remove at any time
[*]If something doesn't work, check the MelonLoader console/log and post it in the comments
[/list]
