Nine Sols Power Mod

=== SHORT SUMMARY (Nexus "Summary" field, max 250 chars) ===
God Mode, health/attack/gold multipliers, infinite dash and unlimited jades (no computing power limit). Everything toggleable in the Configuration Manager menu, no restart needed.

=== DESCRIPTION (Nexus "Description" field, BBCode) ===
[size=4][b]Nine Sols Power Mod[/b][/size]

A set of gameplay boosters for Nine Sols. Enable only what you want - each option is independent and can be changed while the game is running.

[size=3][b]✨ Features[/b][/size]
[list]
[*][b]God Mode[/b] - you take no damage
[*][b]Health Multiplier[/b] (1x to 50x) - incoming damage is divided by the multiplier
[*][b]Attack Multiplier[/b] (1x to 100x) - multiplies your attack damage
[*][b]Gold Multiplier[/b] (1x to 100x) - multiplies the gold you receive
[*][b]Infinite Dash[/b] - dash resets after each roll/dodge
[*][b]Unlimited Jades[/b] - equip any jade combination without the computing power limit ([b]F8[/b] forces a refresh if a jade doesn't unlock right away)
[/list]

[size=3][b]Configuration[/b][/size]
All options live in the BepInEx config and show up in the [b]Configuration Manager[/b] menu in-game - change them live, no restart needed. The config file is at:
[b]Nine Sols/BepInEx/config/opaaaaaaaaaaaa.ninesols.powermod.cfg[/b]

[size=3][b]📋 Requirements[/b][/size]
[list]
[*][url=https://www.nexusmods.com/ninesols/mods/17]BepInEx and Configuration Manager for Nine Sols[/url]
[/list]

[size=3][b]📦 Installation[/b][/size]
[list=1]
[*]Install BepInEx in the game folder and run the game once
[*]Extract the archive into the game folder (or install with Vortex). The file lands in:
[code]Nine Sols/BepInEx/plugins/NineSolsPowerMod.dll[/code]
[*]Play
[/list]

[size=3][b]Notes[/b][/size]
[list]
[*]Disabling the mod restores the original values - safe to add or remove
[*]If something doesn't work, check [b]BepInEx/LogOutput.log[/b] and post it in the comments
[/list]
