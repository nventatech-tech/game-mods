=== DESCRIPTION (Nexus "Description" field, BBCode) ===
[size=4][b]BepInEx and Configuration Manager for Nine Sols[/b][/size]

BepInEx is the mod loader used by Nine Sols mods such as [b]Nine Sols Power Mod[/b]. This package bundles BepInEx 5 x64 with [b]Configuration Manager[/b], so you can change mod settings in-game. You only need to install it once.

[size=3][b]📦 Installation[/b][/size]
[list=1]
[*]Extract the zip. Copy the [b]contents[/b] of the extracted [b]BepInEx[/b] folder ([b]winhttp.dll[/b], [b]doorstop_config.ini[/b] and the inner [b]BepInEx[/b] folder) into the game folder (the one with [b]NineSols.exe[/b]), usually:
[b]C:\Program Files (x86)\Steam\steamapps\common\Nine Sols[/b]
[*]Run the game once and close it. This creates the remaining [b]BepInEx[/b] subfolders.
[*]Drop mod DLLs into [b]Nine Sols/BepInEx/plugins/[/b]
[/list]

[b]Linux / Steam Deck (Proton):[/b] BepInEx needs the winhttp override to load. In Steam, right click Nine Sols > Properties > Launch Options and set:
[code]WINEDLLOVERRIDES="winhttp=n,b" %command%[/code]

[size=3][b]Configuration Manager[/b][/size]
Press [b]F1[/b] in-game to open the settings window and change any mod's options live.

[size=3][b]Checking it works[/b][/size]
After running the game once, [b]Nine Sols/BepInEx/LogOutput.log[/b] should exist and list the loaded plugins. If it doesn't exist, BepInEx isn't loading - wrong folder, or missing launch option on Linux.

[size=3][b]Recommended[/b][/size]
Playing on an ultrawide monitor (21:9 / 3440x1440)? The game needs [url=https://github.com/p1xel8ted/UltrawideFixes/releases]Nine Sols Ultrawide Fix by p1xel8ted[/url] to render correctly - drop its dll into [b]BepInEx/plugins/[/b] after installing this pack.
