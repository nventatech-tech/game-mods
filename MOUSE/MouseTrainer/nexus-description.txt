=== MOD NAME (Nexus "Name" field) ===
MOUSE Trainer

=== SHORT SUMMARY (Nexus "Summary" field, max 250 chars) ===
Toggle God Mode, Infinite Ammo, Double Damage, Double Health and Homing Bullets on the fly with F1-F5. Lightweight BepInEx plugin, no permanent overlay - just a quick on-screen notification when you toggle a cheat.

=== DESCRIPTION (Nexus "Description" field, BBCode) ===
[size=4][b]MOUSE Trainer[/b][/size]

Simple in-game trainer for MOUSE: P.I. For Hire. Five independent cheats, each on its own hotkey, with a short on-screen toast when toggled - no permanent overlay cluttering the screen.

[size=3][b]✨ Features[/b][/size]
[list]
[*][b]F1[/b] - God Mode (starts OFF)
[*][b]F2[/b] - Infinite Ammo (starts OFF)
[*][b]F3[/b] - Double Damage (starts ON)
[*][b]F4[/b] - Double Health (starts ON)
[*][b]F5[/b] - Homing Bullets (starts OFF) - every shot, hitscan or projectile, locks onto the enemy closest to your crosshair. Respects line of sight (no shooting through walls) and removes weapon spread while active
[/list]

[size=3][b]📋 Requirements[/b][/size]
[list]
[*][url=https://www.nexusmods.com/mousepiforhire/mods/1]Patch for BepInEx MOUSE P.I. For Hire[/url] (BepInEx 5 x64)
[/list]

[size=3][b]📦 Installation[/b][/size]
[list=1]
[*]Install BepInEx in the game folder and run the game once
[*]Extract the archive into the game folder (or install with Vortex). The file lands in:
[code]MOUSE/BepInEx/plugins/MouseTrainer.dll[/code]
[*]Play
[/list]

[size=3][b]Notes[/b][/size]
[list]
[*]Each cheat is independent - toggling one never touches the others
[*]Turning a cheat off restores the original values (health/damage go back to normal)
[*]No game files are modified - safe to add or remove at any time
[*]If something doesn't work, check [b]BepInEx/LogOutput.log[/b] and post it in the comments
[/list]
