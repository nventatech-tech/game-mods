=== DESCRIPTION (Nexus "Description" field, BBCode) ===
[size=4][b]BepInEx for Cuphead[/b][/size]

BepInEx is the mod loader used by Cuphead mods such as [b]Cuphead Double Assist[/b] and [b]Cuphead UltraWide[/b]. This is the official BepInEx 5.4.23.5 x64 package - uploaded here so everything you need is in one place. You only need to install it once.

[size=3][b]📦 Installation[/b][/size]
[list=1]
[*]Extract the zip into the game folder (the one with [b]Cuphead.exe[/b]), usually:
[b]C:\Program Files (x86)\Steam\steamapps\common\Cuphead[/b]
After extracting you should have [b]winhttp.dll[/b], [b]doorstop_config.ini[/b] and a [b]BepInEx[/b] folder next to Cuphead.exe.
[*]Run the game once and close it. This creates the [b]BepInEx/plugins[/b] and [b]BepInEx/config[/b] folders.
[*]Drop mod DLLs into [b]Cuphead/BepInEx/plugins/[/b]
[/list]

[b]Linux / Steam Deck (Proton):[/b] BepInEx needs the winhttp override to load. In Steam, right click Cuphead > Properties > Launch Options and set:
[code]WINEDLLOVERRIDES="winhttp=n,b" %command%[/code]

[size=3][b]Checking it works[/b][/size]
After running the game once, [b]Cuphead/BepInEx/LogOutput.log[/b] should exist and list the loaded plugins. If it doesn't exist, BepInEx isn't loading - wrong folder, or missing launch option on Linux.

[size=3][b]Updating old mods[/b][/size]
BepInEx 5.4.23+ changed its Harmony library. Mods built for older BepInEx versions are loaded through a compatibility shim (you'll see a "Shimming..." warning in the log) and some may stop working. If a mod breaks after updating, look for a version rebuilt for BepInEx 5.4.23+.
