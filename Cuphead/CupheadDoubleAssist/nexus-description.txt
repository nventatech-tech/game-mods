=== MOD NAME (Nexus "Name" field) ===
Cuphead Double Assist

=== SHORT SUMMARY (Nexus "Summary" field, max 250 chars) ===
Multiplies your damage and health (2x-10x), with optional infinite health and damage. Toggle everything in-game with hotkeys or controller, with on-screen messages. Fully configurable, config reloads live.

=== DESCRIPTION (Nexus "Description" field, BBCode) ===
[size=4][b]Cuphead Double Assist[/b][/size]

Makes the game easier without changing how it plays: multiplies your damage and health, with optional infinite health and infinite damage. Everything can be toggled in-game.

[size=3][b]✨ Features[/b][/size]
[list]
[*]Damage multiplier (2x to 10x)
[*]Health multiplier (2x to 10x)
[*]Infinite health and infinite damage toggles
[*]Hotkeys: F1 toggles the mod, F2 infinite health, F3 infinite damage (rebindable)
[*]Controller: LT+RT+X toggles the mod, LT+RT+A infinite health, LT+RT+B infinite damage
[*]On-screen message whenever you toggle something
[*]Config file changes apply live - no need to restart the game
[/list]

[size=3][b]📋 Requirements[/b][/size]
[list]
[*][url=https://www.nexusmods.com/cuphead/mods/173]BepInEx for Cuphead[/url] (BepInEx 5 x64)
[/list]

[size=3][b]📦 Installation[/b][/size]
[list=1]
[*]Install BepInEx 5 in the game folder and run the game once
[*]Extract the archive into the game folder (or install with Vortex). The file lands in:
[code]Cuphead/BepInEx/plugins/CupheadDoubleAssist.dll[/code]
[*]Play
[/list]

[size=3][b]Configuration[/b][/size]
The config file is created on first run at:
[b]Cuphead/BepInEx/config/opaaaaaaaaaaaa.cuphead.doubleassist.cfg[/b]

You can edit it with the game open - the mod detects the change and applies it in a couple of seconds, showing "Config reloaded" on screen. Multipliers accept 2 to 10. Hotkey toggles are saved back to the file, so it always reflects the current state.

[size=3][b]Enable only what you want[/b][/size]
Damage and health are independent. Want only the damage multiplier and keep your health untouched? Set this in the config:
[code]
[02 - Ativar ou Desativar]
EnableDamage = true
EnableHealth = false
[/code]
The same works the other way around (only health, normal damage). Changes apply live, no restart needed.

[size=3][b]Notes[/b][/size]
[list]
[*]Works on keyboard and XInput controllers (up to 4)
[*]Turning the mod off restores the original damage and health values
[*]If something doesn't work, check [b]BepInEx/LogOutput.log[/b] and post it in the comments
[/list]
